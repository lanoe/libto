TO library migration guide from 4.5.x to 4.6.x¶

The following changes are to be taken into account to update from 4.5.x to 4.6.x.

Configure options (Linux project)¶

Fingerprint APIs have been disabled by default. Then, if you need fingerprint APIs in your project, you now have to explicitly enable these features.

The following configure options are useless because this is now the default setting:

  • –disable-fingerprint

If required, to enable these feature for your project you can use:

  • –enable-fingerprint

See Library configuration with autotools. to properly configure libTO.

TLS helper API¶

size_t and ssize_t have been replaced respectively by uint32_t and int32_t.

Impacted functions are :

  • TO_helper_tls_send_message()
  • TO_helper_tls_handshake_send_func
  • TO_helper_tls_receive_message()
  • TO_helper_tls_handshake_receive_func
